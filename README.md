# الملاحظات الصوتية — PWA

## ملفات المشروع
```
voice-notes-pwa/
├── index.html      ← التطبيق الرئيسي
├── manifest.json   ← إعدادات PWA
├── sw.js           ← Service Worker (وضع بلا إنترنت)
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

---

## طريقة الاستخدام

### الخيار 1 — فتح مباشرة في المتصفح
افتح ملف `index.html` مباشرة في أي متصفح. يعمل فوراً بدون إعداد.

---

### الخيار 2 — تثبيت على الهاتف (موصى به)

#### iPhone / iPad:
1. انقل المجلد إلى خادم ويب أو استخدم Netlify (مجاناً)
2. افتح الرابط في **Safari**
3. اضغط زر المشاركة (Share) ← **"إضافة إلى الشاشة الرئيسية"**
4. سيظهر التطبيق كأيقونة على هاتفك

#### Android (Chrome):
1. افتح الرابط في **Chrome**
2. اضغط القائمة (⋮) ← **"إضافة إلى الشاشة الرئيسية"**
3. أو انتظر البانر التلقائي

---

### الخيار 3 — نشر مجاني على Netlify (دقيقتان)
1. اذهب إلى **netlify.com** وسجّل دخول
2. اسحب مجلد `voice-notes-pwa` كاملاً وأفلته في صفحة Netlify
3. ستحصل على رابط مثل: `https://your-app.netlify.app`
4. افتحه على هاتفك وثبّته

---
